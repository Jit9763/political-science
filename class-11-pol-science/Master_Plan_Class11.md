# Master Plan: Class 11 Political Science Notes Website

## 🎯 Objective
Create a high-quality, interactive notes website for Class 11 Political Science (Hindi Medium) following the 7-step framework for all 18 chapters.

---

## 🏗️ 7-Step Framework for Each Chapter
1.  **Detailed Introduction:** Engaging start to the chapter.
2.  **NCERT Images & Cartoons:** Visual learning from the textbook.
3.  **Comprehensive Theory:** Core concepts in easy language.
4.  **Box Info (क्या आप जानते हैं?):** Interesting facts and deep dives.
5.  **Vocabulary & Timelines:** Key terms and important dates.
6.  **Mega Question Bank:** 1, 2, 4, and 6-mark questions.
7.  **Quick Summary (निचोड़):** Fast revision points.
